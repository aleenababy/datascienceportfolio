The problem

The main goal of this project is to read the hdf4 file into a pandas dataframe which can be used to understand the physics of the model. The HDF4 output file from the KOSMA-Tau PDR model have some hardcoded values which are not reccomendend. However, the routine takes care of the hardcoded values and try to allow you have flexibility whenever possible.

Pre-requisites:

The following are the pre-requisit programs:

  • for reading HDF4 file using python
    • pyhdf
  • to create the dataframes
    • Numpy
    • Pandas
  • for plotting the file
    • Matplotlib
