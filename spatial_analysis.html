Project Overview:

Objective:

Develop a model to comprehend the distribution of Pulsars in the galaxy with the primary goal of identifying regions conducive to fast radio bursts (FRBs). The project involved simulating the distribution of pulsars in the Milky Way, comparing it with the observed distribution, and employing statistical analysis to enhance our understanding of pulsar distribution.

Key responsibilities

Model Development:
Utilized Python programming to create computer codes for simulating the distribution of pulsars in the Milky Way. The model aimed to provide insights into potential regions for fast radio bursts based on pulsar distribution.
Data Analysis:
Conducted statistical analysis on simulated pulsar distributions. Compared the model-generated distribution with the existing observational data of pulsar locations. Implemented Python for analyzing graphical outputs and extracting meaningful insights.
Outlier Detection and Removal:
Developed robust methods for detecting outliers within the simulated pulsar distribution. Implemented strategies to systematically remove outliers, ensuring the model's accuracy and reliability.

Project Impact:

  • Enhanced Understanding: The project contributed to a deeper understanding of pulsar distribution in the Milky Way, offering valuable insights into regions where fast radio bursts could potentially occur.
  • Methodology Development: Implemented innovative methodologies for outlier detection and removal, contributing to the reliability and accuracy of future simulations.

Project Details:

  • Project Duration: 8 Months
  • Tools Used: Python:NumPy, Matplotlib, SciPy, Shell scripting
  • Collaboration: Collaborated under the guidance of Prof. Avinash Deshpande at Raman Research Institute, Bengaluru, India.

Achievements:

  • Successfully developed and implemented a Python-based model for simulating pulsar distributions.
  • Conducted in-depth statistical analysis, comparing simulated data with observed pulsar populations.
  • Developed effective methods for outlier detection and removal, ensuring the accuracy of the model.

Future Enhancements:

  • Continuously refine the model with updated observational data to improve accuracy.
  • Explore the application of machine learning algorithms for predictive modeling in pulsar distribution.
